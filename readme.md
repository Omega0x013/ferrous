# Ferrous
This started out as an endevour to create a bootloader using rust code and rust code alone. I got pretty far but ultimately I decided that using a little bit of assembly might be OK. Which pretty much leads us up to now, I haven't managed much.

### Deps
- Rustc (obviously)
- NASM
- QEMU x86_64

### Building & Running
You can find all the steps in the makefile, but running `make` will work for most linux users.
